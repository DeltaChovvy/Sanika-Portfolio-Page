# Image Placement Guide

## Images Needed

Place the following images in the `public/` folder:

### 1. Hero Section
- **File**: `sanika-profile.jpg`
- **Location**: `public/sanika-profile.jpg`
- **Description**: Circular profile photo of Sanika
- **Recommended Size**: 800x800px (square, will be displayed as a circle)

### 2. Projects Section
- **File**: `project-academic.jpg`
- **Location**: `public/project-academic.jpg`
- **Description**: Mockup image for the Academic Outreach UX Case Study (mobile app screens)
- **Recommended Size**: 1200x1600px (portrait orientation)

### 3. Resume (Optional)
- **File**: `resume.pdf`
- **Location**: `public/resume.pdf`
- **Description**: Your resume PDF file
- **Note**: Update the link in `components/Navigation.tsx` if you have a different resume location

## How to Add Images

1. Simply drag and drop your images into the `public/` folder
2. Make sure the file names match exactly as shown above (case-sensitive)
3. The website will automatically load these images

## Image Formats Supported
- JPG/JPEG
- PNG
- WebP (recommended for best quality and performance)
- SVG (for logos/icons)

## Notes

- If images are not found, the website will show fallback placeholders
- All images will be automatically responsive
- Profile photo will be displayed as a circle
- Project mockup will maintain aspect ratio
